# Flight-Booking-Conversational-Chatbot
A cloud native conversational chatbot built using Dialogflow CX on Google Cloud Platform that enables real time flight search and booking through API integrations, enhanced with data driven intent optimization.

## Features
- Real time flight search via APIs
- Conversational flows using Dialogflow CX
- Improved intent recognition using BigQuery & Vertex AI
- Simple HTML/CSS frontend interface

## Tech Stack
- Google Cloud Platform (GCP)
- Python
- Dialogflow CX
- BigQuery
- Vertex AI
- HTML, CSS

## Future Improvements
- Add payment integration
- Support hotel and cab booking
- Multilingual support
